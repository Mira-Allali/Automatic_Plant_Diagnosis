# Automatic Plant Diagnosis

## Description
Deep Learning and Classical ML comparative study for plant disease classification.

## Methods
- CNN (TensorFlow)
- PCA + SVM
- K-Fold Cross Validation

## Dataset
PlantVillage (Kaggle)

## Results
- CNN Accuracy: 94%
- SVM Accuracy: 65%
